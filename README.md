# Larafeed
Simple blog app with laravel and 

Steps:
- `git clone`
- `cd larafeed`
- `composer install`
- `npm i` or `yarn` 
- `cp .env-example`
- edit your database details
- `php artisan migrate:fresh --seed` to use dummy data
- `php artisan serve` to start php server 
- open localhost:8000 with browser
- ready!
